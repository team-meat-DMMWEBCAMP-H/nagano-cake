概要<br>
「ながのCAKE」のECサイト

機能<br>
会員側<br>
・アカウントを作成し、ログイン、ログアウトを行うことができます。
・商品を選択し購入することができます。
・支払い方法を銀行振込、クレジットカードから選ぶことができます。
・お届け先住所を、アカウントの住所、登録した住所から選ぶことができます。

管理者側<br>
・会員の注文情報やアカウント情報を閲覧することができます。
・注文情報から入金ステータス、製作ステータスの確認、変更ができます。
・商品を新規登録、編集することができます。

必要要件<br>

Gemfile<br>
・Devise
・Rails バージョン6.1.7.3
・Kaminari バージョン1.2.1
・Bootstrap バージョン4.5.2
